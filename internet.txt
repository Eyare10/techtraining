The Internet is a global network of computers connected to each other which communicate through a standardized set of protocols. The internet is made for informations and these moves in form of bits from one device to another with these devies having their respective ip addresses. The transfer of information is made possible through the use of HTTP (Hypertext transfer protocol. The HTTP is a medium that requests information from the web server once a URL is inputed on the browser. this request sends back the information from the web server to the device's browser. The transfer of these informations can be hitched if certain conditions are not met which gives rise to different error codes.
